
# Galaxy 

## Overview
An iOS app for exploring the most fascinating galaxies in the known universe — built with SwiftUI.

---

## Screenshots



---

## Features

- Browse a curated list of galaxies  
- **Detail screen** for each galaxy with a hero image, key statistics, and fun facts
- **Search** by galaxy name, type, or constellation
- Color-coded galaxy type badges (Spiral, Elliptical, Irregular, and more)
- Fully offline — no internet connection required
- System, Light and Dark mode enabled 

## Built With

- **SwiftUI** — declarative UI framework
- **Swift** — 100% Swift codebase
- No third-party dependencies

## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/auracool365/galaxyiOS.git
   ```
2. Open `galaxy.xcodeproj` in Xcode
3. Select a simulator or connected device
4. Hit **Run** ▶

> Requires Xcode 16+ and iOS 17.2+

## Project Structure

```
galaxy/
├── ContentView.swift        # Main list screen
├── GalaxyRowView.swift      # Custom list row component
├── GalaxyDetailView.swift   # Detail screen for each galaxy
├── galaxy_data.swift        # Galaxy model and data
└── Assets.xcassets/         # Galaxy images
```
